## DB Index

- **DB Index 란?** : DB의 테이블 값을 시간/공간적으로 비용이 적게 들게 가져올 수 있도록 만든 데이터 구조. [3]
- **왜 써야 하는가?** : 데이터가 많을 때, 빠르게 값을 가져오고, 공간적으로도 절약할 수 있게 하기 위해 [2]

### MySQL

- Indexing 에 B+ Tree 형태를 사용
- Clustered Index (자동 설정) / Non Clustered Index (개뱔자가 설정) [1]
