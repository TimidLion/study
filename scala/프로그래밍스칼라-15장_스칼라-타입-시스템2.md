#### 15.3 타입 투영
- **투영(proejct)** : 타입 멤버를 현존하는 다른 특정 타입으로 정의해두는 것
- **타입 지정자** : 타입 투영을 짧게 쓴 것
  ```scala
    Int           // scala.type#Int
    scala.Int     // scala.type#Int
    package pkg {
      class MyClass {
        type t    // pkg.MyClass.type#t
      }
    }
  ```
- 뭔가 별로라고 주장하는 글도 있는 것 같다
  - https://lptk.github.io/programming/2019/09/13/type-projection.html
